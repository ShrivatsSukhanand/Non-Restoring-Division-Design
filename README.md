# Non-Restoring-Division-Design
This design performs 4 bit unsigned division using the established Non Restoring algorithm
It is designed on vivado ML version 2025.2.1

Enhanced FSM understanding and honed debugging skills

Output format is a byte, with MSN representing remainder and LSN representing quotient
